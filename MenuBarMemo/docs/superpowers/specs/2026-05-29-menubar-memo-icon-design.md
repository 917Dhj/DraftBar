# MenuBarMemo 图标设计规格

日期：2026-05-29

## 1. 背景

MenuBarMemo 是一个 macOS 菜单栏 Markdown 备忘录 app。核心体验是从菜单栏快速唤出一张可输入的草稿纸，稳定记录中文和 Markdown 文本。

当前工程里的 `AppIcon.appiconset` 仍是空占位，菜单栏状态项暂用 SF Symbol `square.and.pencil`。本规格定义第一版图标方向，用于后续生成 App 图标和菜单栏 template 图标。

## 2. 设计目标

图标需要同时表达三件事：

1. 这是一个菜单栏 app，而不是普通独立笔记窗口。
2. 这是一个可以写东西的备忘录工具。
3. 它支持 Markdown 草稿输入，但 Markdown 不是唯一品牌符号。

设计应保持克制、清晰、macOS 原生感，贴合现有 Liquid Glass 风格。不要做成复杂插画、重拟物纸张、强品牌渐变或过度装饰。

## 3. 核心图形隐喻

采用“菜单栏落下的便签 + 明确书写动作”。

图形元素：

- 顶部短横条：代表 macOS 菜单栏入口。
- 白色便签块：代表从菜单栏拉出的备忘录纸面。
- 斜向笔：代表可立即输入、写作、记录。
- 轻量 `#`：仅在 App 大图标中作为 Markdown 暗示。

设计优先级：

1. 小尺寸先读出“便签 + 笔”。
2. 中大尺寸再读出“从菜单栏落下”。
3. 最大尺寸才出现 Markdown 暗示。

## 4. App 图标规格

App 图标用于 Finder、Launchpad、About 和打包后的 `.app`。

构图：

- 使用 macOS squircle 外轮廓，不做尖角或圆形 app 图标。
- 顶部约 16%-18% 位置放置深色短横菜单栏。
- 便签位于图标中部偏下，从菜单栏下方出现。
- 笔从右上向左下斜跨便签右下区域，笔尖靠近便签上的短线。
- `#` 作为低对比度符号放在便签左上或中左区域，不能抢过笔和便签。

颜色：

- 背景：浅灰蓝到低饱和绿色的柔和过渡，避免高饱和品牌色。
- 便签：接近白色，带很轻的内描边和阴影。
- 菜单栏短横：深灰蓝，不使用纯黑。
- 笔：深灰蓝笔身，浅色笔尖，保持 macOS 系统感。
- 书写线：可使用一条低饱和蓝绿短线，作为轻微视觉焦点。

层次：

- 图标应有轻微玻璃感和深度，但不要模拟复杂反光。
- 阴影只用于区分背景、便签和笔，不做厚重投影。
- 便签边缘和线条在 32px 仍需可读。

## 5. 菜单栏图标规格

菜单栏图标是 template image，需要适配浅色和深色菜单栏。

菜单栏图标只保留三类形状：

1. 顶部短横：表示菜单栏来源。
2. 中部便签块：表示备忘录。
3. 斜向笔：表示写作输入。

不包含：

- `#` Markdown 符号。
- 彩色背景。
- 阴影、渐变、半透明玻璃效果。
- 细小文字线条。

小尺寸要求：

- 在 16px 宽度下仍能看出便签和笔。
- 线条必须足够粗，不能依赖 1px 细节表达语义。
- 形状应尽量闭合、简洁，便于作为 template image 渲染。

当前 `square.and.pencil` 可以继续作为临时占位，但正式图标应改为同源自定义 template 图标。

## 6. 输出资源

后续制作需要输出：

- `Assets.xcassets/AppIcon.appiconset` 所需 macOS app icon PNG 尺寸。
- 一份可编辑源文件，建议 SVG 或矢量 PDF。
- 菜单栏 template PDF 或 PNG 资源，命名应明确，例如 `StatusBarIcon`.
- 如使用 PDF template，确保在代码中设置 `isTemplate = true`。

App icon asset 应覆盖当前 `Contents.json` 中列出的 macOS 尺寸：

- 16x16 `1x` 和 `2x`
- 32x32 `1x` 和 `2x`
- 128x128 `1x` 和 `2x`
- 256x256 `1x` 和 `2x`
- 512x512 `1x` 和 `2x`

## 7. 验收标准

视觉验收：

1. 1024px 下能看出菜单栏、便签、笔和轻量 Markdown 暗示。
2. 128px 下仍能看出菜单栏落下便签和笔。
3. 32px 下优先读出“备忘录/写作工具”。
4. 16px 菜单栏图标能读出“便签 + 笔”，不依赖 `#`。
5. 浅色和深色菜单栏下 template 图标都清晰。

产品验收：

1. 图标不应像通用 Apple Notes clone。
2. 图标不应像纯 Markdown 编辑器或代码工具。
3. 图标应让用户理解这是一个从菜单栏快速打开的写作/备忘录 app。

工程验收：

1. `AppIcon.appiconset` 中不再只有空占位。
2. 菜单栏状态项使用同源自定义 template 图标。
3. 浅色、深色、不同 scale 下图标没有模糊、错位或过细。

## 8. 非目标

本阶段不设计：

- 完整品牌系统。
- App Store 营销图。
- 启动画面。
- 设置页中的品牌插画。
- 多套主题色。

图标只服务第一版可用 app 的识别和一致性。
